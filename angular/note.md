## Initial
```
npm install -g @angular/cli
ng new my-app
cd my-app
ng serve --open
```

## Commands
- \+ New Component
```
ng generate component xyz
```

- \+ Angular Material
```
ng add @angular/material
```

- \+ Add Dependency
```
ng add _____
```

- \+ run and Watch Tests
```
ng test
```

- \+ Build for Production
```
ng build --prod
```
